# Contributing to ConceptualCommons / Forest Protocol

## 1. Implementers Wanted

Skills needed: DHT/IPFS, zk-proof, vector search  
→ Open an Issue and say hello.

## 2. Genesis-Node Curators Wanted

No coding required.  
Please select fragments using these criteria:

- Public domain only
- Short — one sentence to one paragraph
- Contains a question or provokes thought
- Aim for diversity across culture, era, and theme

Recommended sources: Wikisource, Project Gutenberg  
Submission: Pull Request or Issue

## 3. Join the Discussion

Let's talk about unresolved questions, philosophy, and design.  
→ GitHub Discussions

## このプロジェクトへの参加方法

## 1. 実装者を探しています

DHT/IPFS, zk-proof, ベクトル検索の経験者  
→ Issueで話しかけてください

## 2. genesis-nodesのキュレーターを探しています

プログラムの知識は不要です。  
以下の基準で断片を選んでください：

- パブリックドメインであること
- 一節・一段落程度の短さ
- 問いを含むか、思考を刺激するもの
- 文化・時代・テーマが多様になるよう意識する

推薦先：Wikisource, Project Gutenberg  
提出方法：Pull Request or Issue

## 3. 思想の議論に参加してください

未解決問題、哲学、設計について話しましょう。  
→ GitHub Discussions
